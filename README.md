# PawPal+ (Module 2 Project)

PawPal+ is a smart pet care scheduling system that helps a busy pet owner organize daily care tasks. The project uses Python object-oriented programming to model owners, pets, tasks, and scheduling logic.

## Features

- Add pets to an owner profile
- Add care tasks for each pet
- Store task duration and priority
- Sort tasks by priority
- Filter tasks based on available time
- Generate a clear daily schedule
- Mark tasks as complete
- Test core scheduling behaviors with pytest

python main.py
